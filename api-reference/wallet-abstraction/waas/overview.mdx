Particle Network’s **Onboarding suite** enables developers to easily integrate wallet creation and login flows for end users — especially those unfamiliar with crypto — by combining **Web2-style social logins with Web3-native account creation**.

Under the hood, Onboarding is a **Wallet-as-a-Service product** that creates **non-custodial EOAs**, and optionally pairs them with **smart contract accounts** to support Account Abstraction (AA). This modular system ensures that every app can provide:
- A familiar login UX (email, social, phone)
- Application-embedded wallet access
- Optional advanced functionality via AA (like gasless transactions or custom security)

This suite is tightly integrated with the rest of the Particle stack, especially **Account Abstraction and Smart Wallet infrastructure**.

Particle Network offers two main packages for integrating Web3 functionalities into your dApp:

### Particle Connect
An all-in-one solution that provides:
- Social login integration
- Traditional wallet connections
- Account Abstraction capabilities
- Unified interface for all authentication methods

## Account Abstraction (AA) Overview

Account Abstraction enhances user experience by:

1. **Smart Contract Wallets**: Replaces traditional EOAs with programmable accounts
2. **Flexible Authentication**: Supports logins beyond private key management
3. **Programmable Security**: Add recovery options, session limits, and custom rules
4. **Gas Abstraction**: Enables gasless transactions via paymasters

***

## Modular Implementation

Particle’s AA stack is built to be **modular and configurable**, allowing developers to:
- Use their preferred smart account implementation
- Choose custom bundlers and paymasters
- Integrate with any frontend or auth flow
- Build on existing Web2 infrastructure
